Теорема об ABC-тройках

О чем собственно, теорема? Как гласит Википедия, для взаимно-простых чисел (не имеющих общих делителей) \(a, b, с\), таких что \(a + b = c\), для любого \(\varepsilon > 0\) существует ограниченное число троек abc, таких что:

$$c > rad(abc)^{1 + \varepsilon},$$ где \(rad(abc)\) - радикал произведения a*b*c.

Примеры:

\(a = 100\), \(b = 27\): \(c = a + b = 100 + 27\). Посчитаем радикал: \(rad(abc) = rad(a) * rad(b) * rad(c) = rad(100) * rad(27) * rad(127) = rad(2*2*5*5) * rad(3*3*3) * rad(127) = (2 * 5) * (3) * (127) = 3810\). Если даже взять \(\varepsilon = 0.0001\), то \( 3810^{1 + 0.0001} > 127\), а значит тройка \((100 + 27 = 127)\) не является тройкой ABC.

А вот тройка \((49 + 576 = 625)\) условие теоремы выполняется и она является тройкой ABC (Попробуйте проверить сами).
